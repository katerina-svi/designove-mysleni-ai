# Designové myšlení a AI

Interaktivní vizualizace propojující metodiku Double Diamond s praktickým workflow pro prototypování pomocí LLM.

Součást kurzu **CŽV AI v praxi** - KISK FF MUNI, jaro 2026.

## Obsah

Čtyři fáze Double Diamond aplikované na vibecoding a prototypování s AI:

- **Discover** - sběr dat, nahrání kontextu do AI, analýza výzkumu
- **Define** - tvorba systémových promptů, instructions, knowledge base, PRD
- **Develop** - ideace s AI, customer journey, service blueprint, vibecoding prototypu
- **Deliver** - prezentace, uživatelské testování, iterace, reflexe

## Použití

Stránka funguje jako standalone HTML bez externích závislostí. Podporuje light i dark mode.

Lze embedovat do IS MUNI pomocí iframe:

```html
<iframe src="https://[username].github.io/designove-mysleni-ai/" width="100%" height="600" frameborder="0"></iframe>
```
